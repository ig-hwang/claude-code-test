# 국토교통부 실거래가 API 키 발급 방법

## 1. 공공데이터포털 회원가입

1. [공공데이터포털](https://www.data.go.kr) 접속
2. 회원가입 (본인인증 필요)

## 2. API 신청

1. 로그인 후 검색창에 **"아파트 매매 실거래가"** 검색
2. **"국토교통부_아파트매매 실거래 자료"** 클릭
3. **"활용신청"** 버튼 클릭
4. 활용목적: "개인 학습 및 분석" 입력
5. 신청 완료

## 3. API 키 발급 대기

- 일반적으로 **즉시 발급** 또는 **1-2시간 이내** 승인
- 승인 상태: 마이페이지 > 오픈API > 개발계정 에서 확인

## 4. API 키 확인

1. 마이페이지 > 오픈API > 개발계정
2. **일반 인증키(Encoding)** 복사 (NOT Decoding 키)

## 5. 프로젝트에 적용

```bash
# backend/.env 파일 수정
MOLIT_API_KEY=여기에_복사한_API_키_붙여넣기
```

## 6. 서버 재시작

```bash
# 백엔드 서버 재시작 (터미널에서 Ctrl+C 후)
cd backend
source venv/bin/activate
uvicorn app.main:app --reload
```

## 참고사항

- **무료** 서비스입니다
- 일일 트래픽 제한: 1000건 (충분함)
- 캐싱을 사용하므로 API 호출 최소화됨
- 실거래가 데이터는 1-2개월 전 데이터까지 제공 (법적 공개 시차)

## API 키 발급이 어려운 경우

API 키 없이도 Mock 데이터로 테스트 가능하지만, 실제 시세 분석을 위해서는 API 키 발급을 권장합니다.
